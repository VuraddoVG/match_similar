For start:
    
    npm i

Then

    npm start

    http://127.0.0.1:8080/

In case of
http-server : File C:\Users\%USER%\AppData\Roaming\npm\http-server.ps1 cannot be loaded        
because running scripts is disabled on this system.

Use:

    Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser
